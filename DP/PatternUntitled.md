# Dynamic Programming Roadmap

| Pattern                       | Topics                                      | Important Questions                                                                                                                                        |
| ----------------------------- | ------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1. DP Basics**              | Memoization, Tabulation, Space Optimization | Fibonacci Number, Climbing Stairs, Min Cost Climbing Stairs                                                                                                |
| **2. 1D DP**                  | Single state                                | House Robber, House Robber II, Delete and Earn                                                                                                             |
| **3. Grid DP**                | Paths, Obstacles                            | Unique Paths, Unique Paths II, Minimum Path Sum, Triangle                                                                                                  |
| **4. Subsequence DP**         | Take / Not Take                             | Subset Sum Problem, Partition Equal Subset Sum, Target Sum, 0/1 Knapsack                                                                                   |
| **5. String DP**              | Matching                                    | Longest Common Subsequence, Longest Palindromic Subsequence, Edit Distance, Distinct Subsequences                                                          |
| **6. LIS Pattern**            | Increasing subsequences                     | Longest Increasing Subsequence, Number of Longest Increasing Subsequence, Largest Divisible Subset                                                         |
| **7. Stock DP**               | Buy/Sell                                    | Best Time to Buy and Sell Stock II, Best Time to Buy and Sell Stock III, Best Time to Buy and Sell Stock IV, Best Time to Buy and Sell Stock with Cooldown |
| **8. Partition DP**           | Split intervals                             | Matrix Chain Multiplication, Burst Balloons, Minimum Cost to Cut a Stick                                                                                   |
| **9. Interval DP**            | Ranges                                      | Palindrome Partitioning II, Strange Printer                                                                                                                |
| **10. Digit DP (Advanced)**   | Digits                                      | Count Numbers with Unique Digits, Numbers At Most N Given Digit Set                                                                                        |
| **11. Bitmask DP (Advanced)** | State Compression                           | Partition to K Equal Sum Subsets, Shortest Path Visiting All Nodes                                                                                         |
| **12. Tree DP**               | DFS + DP                                    | House Robber III, Binary Tree Cameras                                                                                                                      |

# Best Order to Learn

### Phase 1 (Beginner)

- ✅ Fibonacci
- ✅ Climbing Stairs
- ✅ Frog Jump
- ✅ House Robber
- ✅ House Robber II

**Goal:** Learn memoization, tabulation, and space optimization.

---

### Phase 2 (Grid DP)

- Unique Paths
- Unique Paths II
- Minimum Path Sum
- Triangle
- Maximum Falling Path Sum

---

### Phase 3 (Take / Not Take Pattern)

- Subset Sum
- Partition Equal Subset Sum
- Count Subsets
- Target Sum
- 0/1 Knapsack
- Unbounded Knapsack
- Coin Change
- Coin Change II
- Rod Cutting

---

### Phase 4 (String DP)

- Longest Common Subsequence (LCS)
- Print LCS
- Longest Common Substring
- Longest Palindromic Subsequence
- Edit Distance
- Distinct Subsequences
- Wildcard Matching

---

### Phase 5 (LIS Pattern)

- Longest Increasing Subsequence
- Print LIS
- Largest Divisible Subset
- Longest String Chain
- Longest Bitonic Subsequence
- Number of LIS

---

### Phase 6 (Stock DP)

Learn all stock variants in order:

1. Stock I
2. Stock II
3. Stock III
4. Stock IV
5. Cooldown
6. Transaction Fee

---

### Phase 7 (Partition DP)

- Matrix Chain Multiplication
- Minimum Cost to Cut Stick
- Burst Balloons
- Boolean Parenthesization

---

### Phase 8 (Advanced)

- Tree DP
- Digit DP
- Bitmask DP

# If You Want to Master DP for Interviews

This sequence covers the vast majority of interview questions:

1. DP Basics (5 problems)
2. 1D DP (5 problems)
3. Grid DP (5 problems)
4. Knapsack / Subsequence DP (10 problems)
5. String DP (10 problems)
6. LIS Pattern (6 problems)
7. Stock DP (6 problems)
8. Partition DP (6 problems)
9. Tree DP (3 problems)
10. Digit DP & Bitmask DP (optional, 5–8 problems)

**Total:** Around **60–70 carefully chosen problems** are enough to build a strong DP foundation.
